# Installation

## Requirements

- **Minimum Node.js version**: 22.0.0
- **Recommended version**: See `.node-version` file in the project root

## Install the Package

```bash
npm install --save-dev @cawpea/coderef
```

You can also use yarn or pnpm:

```bash
# yarn
yarn add --dev @cawpea/coderef

# pnpm
pnpm add --save-dev @cawpea/coderef
```
